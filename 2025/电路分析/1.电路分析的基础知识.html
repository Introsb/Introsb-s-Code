1.1 电路和电路模型

一. 电路分析

  1. 激励:电源或信号源的电压或电源
  2. 响应:由于激励在电路各部分产生的电压或电源
  3. 注意:电压源的电压是激励，它的电流随外电路不同而不同，所以电压源的电流为响应

二. 电路原件的分类

1.电路元件的图形符号

2.线圈的几种电路模型

3.电路的结构

  1. 支路(branch):各条支路流过不同的电流
  2. 节点(node):三个或三个以上支路的连接点
  3. 回路(loop):由支路所构成的闭合路径
  4. 网孔(mesh):平面电路中,内部不含支路的回路
  5. 一端口(one-part):又称为二端网络,向外引出一对端子的电路
  6. 计数b,n,l,m

三.元件的连接

1.开路(Open Circuit)

  1. 电路断开,断开处电流为0
  2. 处于开路的两点间电压通常不为0,其大小由电路的其他部分决定, $u_{oc}$ 表示开路电压

2.短路(Short Circuit)

  1. 电路中的两点用导线连接,两点间的电压为0

  2. 处于短路的两点间电流通常不为0,其大小由电路的其他部分决定, $i_{sc}$ 表示短路电流

    **说明:直流稳态**电路中,电容相当于开路,电感相当于短路

四.电压和电流的方向

1.参考方向的表示方法

  1. 电压的方向为高电位指向低电位的方向
  2. 电流的方向为正电荷运动的方向
  3. 关联参考方向:电流的参考方向与电压的参考方向一致
  4. 画图参考:

2.电位

  • 电路中任意点与参考点之间的电压
  • 参考点的电位为 0 ,一般用接地符号"⏚"表示

五.能量和电功率

1.不同参考方向的功率情况

  • 电功率: $$ P = UI $$
  • 以关联参考方向为例:
  • p > 0: 吸收功率
  • p < 0: 发出功率

2.电路元件在电路中的实际作用

  • 负载:电压电流的实际方向一致
  • 电源:电压电流的实际方向相反

3.功率平衡

$$p_{\text{发出}} = p_{\text{吸收}}$$

六.线性无源元件

1.电阻

  • VCR(Voltage Current Relation):在电压和电流的关联方向下: $$ u = i R $$
  • 电阻的电功率: $$ P = ui = i^2 R = \frac{u^2}{R} $$
  • 电导:$$ G= \frac{1}{R} (单位 : S, 西门子) $$

2.电容

  • 定义:储存电荷的器件,具有储存电场能量的特性
  • 线性电容: $$ C = \frac{q}{u} $$
  • 电容的特性方程(VCR):

$$ \begin{aligned} i &= \frac{d~q}{d~t} \ i &= c \frac{d~u}{d~t} \end{aligned} $$

  • 电容元件储存的电场能量:$$ W_{C}(t)=\frac{1}{2}Cu^{2}(t) $$

3.电感

  • 磁通链（(\Psi)）和电流（(i)）的比值即电感量（(L)）可以表示为：

[ L = \frac{\Psi}{i} ]

  • 电感元件的特性方程（VCR）

[ u = L \frac{di}{dt} ]

  • 电感元件储存的磁场能量

[ W_L = \frac{1}{2} Li^2(t) ]

七.独立电源

1.电压源

特点:

  • 电压 ( u(t) ) 的函数是固定的，不随外电路改变
  • 电流随外电路的不同而变化

特殊情况:

  • 电压为零的电压相当于短路

2.电流源

特点:

  • 电流 (i(t)) 的函数是固定的，不会因外电路的不同而改变
  • 电压随外电路的不同而变化

特殊情况:

  • 电流为零的电流源相当于开路

八.受控电源

九.基尔霍夫电流定律（KCL）

1. 内容

任何时刻，对任一节点，所有与之相连支路电流的代数和恒等于零

2. 公式

[ \sum i = 0 ]

等价形式：
[ \sum i_{\text{in}} = \sum i_{\text{out}} ]

3. 说明

  • 规定流入节点的电流取“+”号，流出取“−”号
  • 流向由电流的参考方向判断

依据：电流的连续性
